JavaScript Switch

The switch statement is a control structure used to execute different blocks of code based on the value of a given expression. It compares the expression against multiple case labels, and when a match is found, the corresponding block of code is executed. Each case typically ends with a break statement to prevent fall-through to the next case. If none of the cases match, the default block is executed, serving as a fallback option.
